# FocusFlow

A simple Todoist / Full Focus Planner style task app.

It runs from GitHub Pages and saves tasks in your browser using local storage.

## How to put this on GitHub

1. Go to https://github.com
2. Click **New repository**
3. Name it `focusflow`
4. Choose **Public** or **Private**
5. Click **Create repository**
6. Click **uploading an existing file**
7. Upload `index.html` from this folder
8. Click **Commit changes**

## Turn on GitHub Pages

1. In your new GitHub repository, click **Settings**
2. Click **Pages** in the left menu
3. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ root**
4. Click **Save**
5. Wait 1-3 minutes
6. GitHub will show a link like:
   `https://yourusername.github.io/focusflow/`

## Make it feel like a desktop app

Open the GitHub Pages link in Chrome or Edge.

### In Edge
1. Click the three dots in the upper right
2. Click **Apps**
3. Click **Install this site as an app**
4. Name it **FocusFlow**
5. Pin it to your taskbar

### In Chrome
1. Click the three dots in the upper right
2. Click **Save and share**
3. Click **Create shortcut**
4. Check **Open as window**
5. Name it **FocusFlow**

## Important

Your tasks are saved in the browser you use. If you open it on a different computer or a different browser, it will not have the same tasks.

Use **Export backup** occasionally to download a backup JSON file.
