# FocusFlow UI Upgrade

Upload this `index.html` to your existing GitHub repository, replacing the current `index.html`.

## What changed
- Cleaner layout
- Better mobile/tablet responsiveness
- Edit button for tasks
- Import/export backup
- Improved empty states
- More reliable saving using browser local storage

## Upload steps
1. Go to your `focusflow` repository on GitHub.
2. Click `index.html`.
3. Click the pencil/edit icon.
4. Select all existing text and delete it.
5. Open the new `index.html` from this ZIP in Notepad.
6. Copy everything and paste it into GitHub.
7. Click `Commit changes`.
8. Wait about one minute, then refresh your FocusFlow app.
